# StremFusion v1.2.0

Professional Android & iOS Device Mirroring and Management.

## Installation
- For a permanent installation, run `StremFusion Setup 1.2.0.exe`.
- For quick use without install, run `StremFusion-v1.2.0-Portable.exe`.

Please refer to `QUICK_START_GUIDE.md` for detailed instructions on enabling USB debugging and WiFi mirroring.
